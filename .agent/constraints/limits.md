## CONSTRAINTS (Local Worker)
- **No Frontend:** Do not install React, Next.js, or any UI library[cite: 1].
- **No Web Server:** The application is a CLI background worker, not a server[cite: 1].
- **Hardware Use:** Optimized for local GPU/CPU rendering during low-activity hours (night)[cite: 1].
- **Security:** Local files must not be exposed; only the final URL is sent to the cloud[cite: 1].
